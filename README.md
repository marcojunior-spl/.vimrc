INSTALLATION INSTRUCTIONS
=========================

The provided file is in .txt format. Please follow these steps to install the configuration:

1. Copy the content from the source .txt file.

2. Paste it into your .vimrc file.
   IMPORTANT: Overwrite any existing content (delete all old lines in your .vimrc first).

3. Open Vim and enter command mode to run the following command:
   
   :PlugInstall

4. Afterwards, reload the configuration by running:

   :source %

   NOTE: You can safely ignore any error messages that appear during this last step.
